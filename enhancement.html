Enhancements

This page describes the additional enhancements implemented in our job search portal.

1. CSS Animations & Transitions

We have implemented smooth hover effects on buttons and links using CSS transitions. This improves the user experience by providing visual feedback.

Code Used:

button:hover {
    background-color: #4CAF50;
    transition: background-color 0.3s ease-in-out;
}

Example applied: Apply Page

Reference: MDN CSS Transitions

2. Responsive Design

We have used CSS media queries to make the website mobile-friendly. The layout adjusts based on the screen size for better readability.

Code Used:

@media (max-width: 768px) {
    body {
        font-size: 14px;
    }
    nav ul {
        display: block;
        text-align: center;
    }
}

Example applied: Home Page

Reference: MDN Media Queries
