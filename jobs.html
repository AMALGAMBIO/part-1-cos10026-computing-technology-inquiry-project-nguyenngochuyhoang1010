DevOps Engineer

Reference Number: DE003

Job Title: DevOps Engineer

Job Summary: Responsible for designing, implementing, and maintaining the infrastructure and tools needed for continuous integration, delivery, and deployment of software applications.

Salary Range: VND 20,000,000 to VND 60,000,000

Job Responsibilities

  • Design and manage CI/CD pipelines to automate the build, test, and deployment processes.
    Monitor and optimize system performance, scalability, and reliability.
    Collaborate with development and operations teams to ensure seamless integration and deployment.
    Implement and manage infrastructure as code (IaC) using tools like Terraform or CloudFormation.
    Configure and maintain cloud infrastructure (e.g., AWS, Azure, GCP).
    Ensure system security by implementing best practices and monitoring for vulnerabilities.
    Troubleshoot and resolve issues related to infrastructure, deployment, and performance.

Qualifications

Essential Skills:

  1. Proficiency in CI/CD tools like Jenkins, GitLab CI/CD, or CircleCI.
    Experience with containerization and orchestration tools like Docker and Kubernetes.
    Strong knowledge of cloud platforms such as AWS, Azure, or Google Cloud.
    5+ years of experience in DevOps or a related field.
    Familiarity with scripting languages like Python, Bash, or PowerShell.

Nice-to-Have Skills:

  1. Experience with infrastructure as code (IaC) tools like Terraform or Ansible.
    Knowledge of monitoring and logging tools like Prometheus, Grafana, or ELK Stack.
    Understanding of networking concepts and security best practices.
    Certifications in cloud platforms (e.g., AWS Certified DevOps Engineer, Azure DevOps Engineer Expert).
